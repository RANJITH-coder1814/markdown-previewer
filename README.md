# markdown-previewer
A simple web application that allows users to write Markdown text and instantly see the formatted HTML preview. It updates the preview in real-time as the user types.
# Markdown Previewer

A simple Markdown Previewer built using HTML, CSS, and JavaScript.

## Features
- Live Markdown preview
- Real-time updates
- Simple and clean interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Marked.js

## How to Run
1. Clone the repository
2. Open index.html in your browser
